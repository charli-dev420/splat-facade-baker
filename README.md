# Splat Facade Baker

Temporary bootstrap repository.

The full project is ready as a Git bundle. Because the user currently has no PC access, this repo includes a mobile bootstrap workflow:

1. Download `splat-facade-baker-private-ready-v2.9.bundle` from ChatGPT.
2. Upload it to this repository at:

```text
bootstrap/splat-facade-baker-private-ready-v2.9.bundle
```

3. Commit directly to `main`.
4. GitHub Actions will run `Bootstrap repository from uploaded git bundle` and force-push the real repository contents plus tags.

See:

```text
bootstrap/README.md
```
