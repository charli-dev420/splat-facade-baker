# Mobile bootstrap

This repository is temporarily bootstrapped for a no-PC setup.

## Goal

Upload the Git bundle generated by ChatGPT into this `bootstrap/` folder. A GitHub Action will automatically clone that bundle and force-push the real project contents to `main`, including tags.

## Required file

Upload this file here:

```text
splat-facade-baker-private-ready-v2.9.bundle
```

Expected path after upload:

```text
bootstrap/splat-facade-baker-private-ready-v2.9.bundle
```

## Mobile steps

1. Download the bundle from ChatGPT.
2. Open this repository in GitHub mobile web.
3. Go to the `bootstrap/` folder.
4. Use `Add file` / `Upload files`.
5. Upload `splat-facade-baker-private-ready-v2.9.bundle`.
6. Commit directly to `main`.
7. Open the `Actions` tab and wait for `Bootstrap repository from uploaded git bundle`.

After success, this temporary bootstrap folder and workflow will be replaced by the real project repository from the bundle.
