# LoRA Clean Render

The clean-render LoRA is intended to teach image generators the technical format needed by SFB: centered object, orthographic feel, neutral solid background, soft neutral light, no cast shadows, no bloom, no cinematic effects and clean silhouettes.

The LoRA should not be treated as a medieval style LoRA. Object/style diversity should come from the prompt and dataset mix; this LoRA only nudges the output into a TRELLIS/TripoSplat-friendly presentation.
