# Product Spec

SFB is an offline production pipeline that converts clean canonical views, maps, or splats into lightweight 2.5D mobile assets.

## Primary user

Technical artist or solo developer building stylized / semi-realistic mobile environments.

## Core promise

Use the best visible face of a controlled image/splat, discard the unreliable 360° hallucination, and export a cheap mobile-ready façade.

## Non-goals

- 360° clean mesh reconstruction.
- Runtime splat renderer.
- General-purpose level editor.
- General-purpose model training platform.
